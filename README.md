# Heartfelt Gifts — Remotion Project

Vertical 1080x1920 Remotion project producing a 30s explainer video "Heartfelt Gifts" (Valentine's Gram concept).

How to preview

1. Install dependencies:

```bash
cd "/Users/codymasumiya/ai social media manager"
npm install
```

2. Run Remotion studio (preview):

```bash
npm run studio
# or
npx remotion studio
```

3. To render a final MP4 (will output to `out/video.mp4`):

```bash
npm run build
```

Notes
- All important content is constrained to the SAFE ZONE (150px top, 170px bottom, 60px sides).
- Font: Inter (400,600,800). Uses `@fontsource/inter`.
- Placeholder music included as a placeholder track (not licensed).
- Voiceover support: the project includes captions and a placeholder voiceover API hook; replace with your audio file as needed.
